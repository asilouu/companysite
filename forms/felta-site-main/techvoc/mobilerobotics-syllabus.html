Course Syllabus

Course Overview

Mobile Robotics (MROB-101) is an introductory course that covers the fundamental principles and practical applications of mobile robotics. Students will learn about robot navigation, sensor integration, motion planning, and control systems through hands-on projects and laboratory exercises.

Basic Competencies

Students will develop the following fundamental competencies through practical exercises and hands-on training:

  • Use hand tools for robot assembly and maintenance
  • Perform mensuration and calculation for robot components and movements
  • Prepare and interpret technical drawings for robot design and assembly
  • Apply quality standards in robot construction and testing
  • Perform computer operations for robot programming and control
  • Terminate and connect electrical wiring and electronics circuits
  • Test electronic components for robot systems

Sample Tasks:

  • Assemble a basic mobile robot chassis using appropriate hand tools
  • Calculate and verify robot wheel circumference and gear ratios
  • Create and interpret wiring diagrams for sensor integration
  • Perform quality checks on assembled robot components
  • Program basic robot movements using computer software
  • Connect and test motor drivers and sensors
  • Verify functionality of electronic components using multimeters

Course Objectives

By the end of this course, students will be able to:

  • Understand fundamental concepts of mobile robotics and autonomous systems
  • Implement basic navigation algorithms for mobile robots
  • Integrate and calibrate various sensors for robot perception
  • Develop and test motion planning strategies
  • Apply SLAM (Simultaneous Localization and Mapping) techniques
  • Design and implement control systems for mobile robots
  • Work effectively in teams on robotics projects
